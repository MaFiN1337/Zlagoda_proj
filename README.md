# Restaurant project

How to setup:

1. git clone https://github.com/Solomka/restaurant.git
2. change db properties in:
    - src/main/resources/db.properties
    - src/main/webapp/META-INF/context.xml
3. build war artifact
4. deploy war artifact to Tomcat
